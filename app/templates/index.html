Plantopia - A Vegetation Cover Monitoring Application



Plantopia is a project build to estimate, monitor and analyze change in vegetation cover using GIS techniques. Vegetation cover defines the percentage of soil which is covered by green vegetation. In a wide range of environments, vegetation cover greatly affects both surface water runoff and sediment loss. A vegetation cover of 45-50% is considered a critical value since above this value soils are adequately protected from raindrop impact and soil erosion is significantly reduced. This project aims at the monitoring of the changing land cover and thus creating awareness regarding it.

The features of each page are as followed:

This page provides the user options to classify land cover based on multiple methods. The first method includes using the MODIS dataset to divide the landcover into multiple classes and rendering the results in a interactable map. The second method is a more complex one where a manually selected dataset was created to get an idea of the various classes and an ML based classification algorithm was used.

This page provides the user the ability to generate spectral change maps that will be visualized instantly post selecting the start and end year for change detection. The generated spectral map will be added to the map itself.

This page provides the user with the ability to generate vegetation change timelapses via manually selecting a region and a time period. This timelapse image will be added to the map and can be exported to get the final result as a GIF image that can easily be shared.
